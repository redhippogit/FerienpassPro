Warum jetzt

Die aktuelle Ferienpass-Plattform lässt alle drei Nutzergruppen im Stich: Eltern kämpfen mit fragmentierten Informationen, Anbieter verwalten alles manuell, das Ferienpass-Team hat keinen Überblick. goSNOB.ai liefert die Infrastruktur in 15 Wochen — dreisprachig, KI-gestützt.

Rahmenbedingungen

Was wir voraussetzen.
Was der Kunde liefert.

Kritisch: vor Woche 1–2 klären

Verzögerungen bei Stripe-Zugang oder Hosting-Entscheid verschieben den Go-Live direkt. Diese Punkte sind nicht optional.

Stripe-Zugang

Zugang (Test + Prod) ab Projekt-Start erforderlich. Ohne Stripe kein Zahlungsflow möglich.

Hosting-Entscheid Wo 1–2

Betrifft gesamte Architektur: DB, Deployment, File Storage. Empfehlung: Vercel / Supabase.

Drittanbieter-Konten

Kunde erstellt Konten für E-Mail-Delivery (SendGrid/Mailgun), Image-Hosting. goSNOB.ai erhält Zugang.

Zahlungsmodell

Ferienpass zentralisiert alle Zahlungen. Anbieter-Auszahlungen offline. Marketplace Payouts = Phase 2.

Kein Datenmigration

Greenfield-Plattform. Anbieter registrieren sich selbst. Initiale Inhalte via Admin-Modul.

Copy vom Kunden

Statische Inhalte (Homepage, About, FAQ) werden vom Kunden auf DE geliefert. goSNOB.ai adaptiert für EN/FR.

Kursinhalt-Sprache

Kursbeschreibungen in Originalsprache des Anbieters. Platform-UI dreisprachig. Auto-Übersetzung = Phase 2.

Subventions-Validierung

Rabatte offline abgewickelt. Digitale Validierung beim Checkout = separater Scope.

Hosting-Kosten Post-Launch

Cloud-Kosten gehen auf den Kunden. Geschätzt: unter CHF 200/Monat bei Launch-Scale.

Ein Environment

Kein separates Dev/Staging/Prod-Setup. Ein Staging → Production bei Go-Live.

Registrierungspflicht

Jede Aktion über Browsen hinaus erfordert Login. Kein Guest Checkout im Scope.

CI/CD via GitHub Actions

Deployment via GitHub Actions mit platform-nativem Deployment. Kein Custom-Infrastructure-Management.

FAQ

Kann ich Add-ons nach Vertragsabschluss noch hinzufügen?

+
Add-ons müssen grundsätzlich vor Vertragsabschluss definiert werden, da sie Scope und Timeline beeinflussen. Einige (z.B. XLSX-Export) können später separat beauftragt werden — in Absprache und gegen Aufpreis.

Was passiert ohne Stripe-Account?

+
Ein Stripe-Account muss vor Projekt-Start eingerichtet und goSNOB.ai zugänglich gemacht werden. Ohne Stripe-Zugang kann kein Zahlungsflow gebaut werden.

Wer schreibt die Inhalte?

+
Der Kunde liefert alle statischen Seiteninhalte auf Deutsch. goSNOB.ai implementiert und adaptiert für EN/FR. Content-Erstellung ist kein Teil des Scopes.

DSGVO / DSG (Schweiz)?

+
Plattform wird DSGVO-konform gebaut: Datenminimierung, Consent-Management, sichere Verarbeitung. Daten Minderjähriger besonders geschützt. Rechtliche Beratung liegt beim Kunden.
