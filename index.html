Ausgangslage

Das Problem
ist real.

Warum jetzt

Die aktuelle Ferienpass-Plattform laesst alle drei Nutzergruppen im Stich. goSNOB.ai liefert die Infrastruktur in 15 Wochen — dreisprachig, KI-gestuetzt.

Eltern
Fragmentiert & chaotisch
Dutzende Websites, kein Kalender, keine Reviews, kein TWINT. Buchung via E-Mail 2026.
Kursanbieter
Alles manuell
Kein Self-Service-Portal. Jede Aenderung = Telefonat. Keine Kapazitaetsanzeige.
Ferienpass-Team
Kein Ueberblick
Kein Dashboard, keine automatisierte Kommunikation, kein Approval-System.

Rahmenbedingungen

Was wir voraussetzen.
Was der Kunde liefert.

Kritisch: vor Woche 1–2 klaeren

Verzoegerungen bei Stripe-Zugang oder Hosting verschieben den Go-Live direkt.

Stripe-Zugang

Zugang (Test + Prod) ab Projekt-Start. Ohne Stripe kein Zahlungsflow.

Hosting-Entscheid Wo 1–2

Betrifft Architektur, DB, Deployment. Empfehlung: Vercel / Supabase.

Drittanbieter-Konten

E-Mail-Delivery (SendGrid/Mailgun) — Kunde erstellt, goSNOB.ai erhaelt Zugang.

Zahlungsmodell

Ferienpass zentralisiert Zahlungen. Anbieter-Auszahlungen offline. Marketplace = Phase 2.

Kein Datenmigration

Greenfield-Plattform. Anbieter registrieren selbst. Inhalte via Admin-Modul.

Copy vom Kunden

Statische Inhalte auf DE vom Kunden. goSNOB.ai adaptiert fuer EN/FR.

Kursinhalt-Sprache

Kursbeschreibungen in Originalsprache. Platform-UI dreisprachig. Auto-Uebersetzung = Phase 2.

Subventions-Validierung

Rabatte offline. Digitale Validierung = separater Scope.

Hosting-Kosten Post-Launch

Cloud-Kosten auf Kunden. Geschaetzt: unter CHF 200/Monat bei Launch-Scale.

Ein Environment

Staging wird zu Production. Kein Dev/Staging/Prod-Setup.

Registrierungspflicht

Jede Aktion erfordert Login. Kein Guest Checkout.

CI/CD via GitHub Actions

Deployment via GitHub Actions. Kein Custom-Infrastructure-Management.

FAQ

Kann ich Add-ons nach Vertragsabschluss noch hinzufuegen?

+
Add-ons muessen grundsaetzlich vor Vertragsabschluss definiert werden. Einige koennen spaeter separat beauftragt werden.

Was passiert ohne Stripe-Account?

+
Ein Stripe-Account muss vor Projekt-Start eingerichtet sein. Ohne Stripe kein Zahlungsflow moeglich.

Wer schreibt die Inhalte?

+
Der Kunde liefert statische Inhalte auf DE. goSNOB.ai adaptiert fuer EN/FR. Content-Erstellung ist kein Teil des Scopes.

DSGVO / DSG (Schweiz)?

+
Plattform wird DSGVO-konform gebaut. Daten Minderjaehriger besonders geschuetzt. Rechtliche Beratung beim Kunden.

Wie kontaktiere ich goSNOB.ai?

+
Alle Anfragen gehen direkt an team@gosnob.com
